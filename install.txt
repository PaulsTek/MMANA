[セットアップの方法]
  セットアップしたいドライブにダウンロードした「MMANA***.EXE」を
コピーし実行します（###はバージョン番号です）。
　ディレクトリ（フォルダ）付きの自己解凍形式ですので、「MMANA」
というディレクトリが作成されてその中にすべて解凍されます。

[How to set up] 
Copy and run the downloaded MMANA***.EXE to the drive you want to set up (### is the version number).　Since it is a self-extracting format with a directory (folder), a directory called "MMANA" is created and all are decompressed in it.

[Execution method] 
"MMANA.EXE" decompressed in the directory "MMANA" is the main program So double-click it to run it (My Computer Or "MMANA.EXE" is displayed as an icon when displayed in Explorer. increase). 
Files with the extension "maa" in the subdirectory "MMANA\ANT\" is a sample antenna definition file.


[How to uninstall] 
Delete all files in the directory "MMANA".

MMANA in the system registry and Windows directory does not create registry files in Uninstall MMANA just delete all MMANA folders (directories) is OK.
